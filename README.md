# node-rdflib
Node.js wrapper for rdflib (https://github.com/RDFLib/rdflib).

#Getting Started

## Install
Github
npm install git+https://github.com/Lambda-IT/node-rdflib

# Features
## Converter Functionality
Supports conversation for the following formats:
- RDF/XML
- N3
- NTriples
- N-Quads
- Turtle
- TriX
- Trig
- RDFa
- JsonLd
